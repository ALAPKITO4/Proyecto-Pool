# Carpeta de Iconos - Pool App

Esta carpeta debe contener los iconos PNG para la PWA.

## Archivos necesarios:

1. **icon-192.png** (192x192 píxeles)
   - Icono cuadrado para app instalada
   - Recomendado: Logo de Pool redimensionado

2. **icon-512.png** (512x512 píxeles)
   - Icono más grande para tienda de apps
   - Recomendado: Logo de Pool en alta resolución

## Cómo crear los iconos:

Si no tienes un logo, puedes:
1. Usar un generador online: https://www.favicon-generator.org/
2. Crear desde tu logo actual
3. Usar los colores de la app (tema color: #FF6B35)

## Nota importante:

Asegúrate de que:
- Sean cuadrados (192x192 y 512x512)
- Estén en formato PNG
- Tengan fondo transparente (recomendado)
- Sean legibles a diferentes tamaños

Una vez agregues los archivos, la PWA se verá con el icono al instalar.
