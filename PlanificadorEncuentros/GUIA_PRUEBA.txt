PRUEBA 2: AGREGAR PERSONAS
============================

✅ Deberías ver:
   - Campo de texto "Escribe un nombre"
   - Botón "Agregar"
   - Botón "Agregar desde WhatsApp"
   - Botón "Continuar" (gris/deshabilitado)

✅ Prueba agregando personas:

   CASO A: Nombre vacío
   • Escribe solo espacios
   • Haz clic "Agregar"
   • RESULTADO: Mensaje de error "El nombre no puede estar vacío"

   CASO B: Nombre corto
   • Escribe "A"
   • Haz clic "Agregar"
   • RESULTADO: Mensaje "El nombre debe tener al menos 2 caracteres"

   CASO C: Nombre válido
   • Escribe "Juan"
   • Haz clic "Agregar"
   • RESULTADO: Aparece etiqueta verde con "Juan" y ✕

   CASO D: Nombre duplicado
   • Escribe "Juan" de nuevo
   • Haz clic "Agregar"
   • RESULTADO: Mensaje "Esta persona ya está en la lista"

   CASO E: Agregar más personas
   • Escribe "María"
   • Haz clic "Agregar"
   • Escribe "Carlos"
   • Presiona Enter (o haz clic Agregar)
   • RESULTADO: Ahora hay 3 personas en la lista

✅ Prueba otras funciones:
   • Haz clic en la ✕ de una persona → se elimina
   • El botón "Continuar" ahora está verde y habilitado
   • Haz clic en "Agregar desde WhatsApp" → se abre WhatsApp

   ANTES DE CONTINUAR: Vuelve a agregar al menos 1 persona

PRUEBA 4: SELECCIONAR HORARIO
==============================

✅ Deberías ver:
   - Campo "Hora de inicio" con selector de hora
   - Campo "Hora de fin" con selector de hora
   - Botón "Continuar" (gris/deshabilitado)

✅ Prueba los horarios:

   CASO A: Horarios válidos
   • Selecciona "14:00" como hora de inicio
   • Selecciona "16:00" como hora de fin
   • RESULTADO: Botón "Continuar" se habilita (verde)
   • No hay mensaje de error

   CASO B: Hora de fin menor que inicio
   • Selecciona "18:00" como hora de inicio
   • Selecciona "17:00" como hora de fin
   • RESULTADO: Aparece error rojo "❌ La hora de fin debe ser posterior a la de inicio"
   • Botón "Continuar" se deshabilita

   CASO C: Horas iguales
   • Selecciona "15:00" como hora de inicio
   • Selecciona "15:00" como hora de fin
   • RESULTADO: Aparece el mensaje de error
   • Botón "Continuar" deshabilitado

   CASO D: Campo vacío
   • Borra ambos campos
   • RESULTADO: Botón "Continuar" deshabilitado

✅ Configura horarios válidos:
   • Hora de inicio: 19:00
   • Hora de fin: 21:00
   • Haz clic "Continuar"
