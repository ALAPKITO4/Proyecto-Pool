⚙️ CÓDIGO JAVASCRIPT
====================

Funciones Principales:
   • goToStep(n) - Navegación entre pasos
   • addPerson() - Agregar persona
   • removePerson(i) - Eliminar persona
   • selectLocation(loc) - Seleccionar ubicación
   • validateTime() - Validar horarios
   • confirmPlan() - Confirmar plan
   • restart() - Reiniciar aplicación

Funciones Auxiliares:
   • updateUI() - Actualizar interfaz
   • updatePersonsList() - Renderizar personas
   • updateSummary() - Mostrar resumen
   • validateCurrentStep() - Validar paso actual
   • saveState() - Guardar en localStorage
   • loadState() - Cargar desde localStorage
   • formatTime() - Formatear hora
   • showError() - Mostrar mensaje error

Características:

   ✓ Manejo de eventos eficiente
